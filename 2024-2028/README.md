# SIG AI Batch 2024–2028

The **SIG AI Batch of 2024–2028** is a group of members with a shared interest in Artificial Intelligence and intelligent systems. This repository serves as the central portfolio for the batch, showcasing projects, research, technical explorations, and contributions

---

## Core Focus Areas

* Machine Learning
* Deep Learning
* Computer Vision
* Natural Language Processing (NLP)
* Large Language Models (LLMs)
* Agentic AI
* Data Science & Analytics
* MLOps & AI Deployment
