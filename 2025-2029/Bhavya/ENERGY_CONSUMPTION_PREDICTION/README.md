# Smart Energy Consumption Predictor

## Problem Statement
Predict daily energy consumption using weather and time-based features.

## Objective
To help users estimate power usage and reduce electricity costs.

## Dataset
UCI Appliances Energy Prediction Dataset

## Tech Stack
Python, Scikit-learn, FastAPI, HTML, CSS, JavaScript

## ML Models Used
Linear Regression, Random Forest, Gradient Boosting

## Evaluation Metrics
MAE, RMSE, R² Score

## Features
- Daily energy prediction
- Cost estimation
- Usage category
- Energy-saving suggestions
- Prediction dashboard

## How to Run
1. Clone repo
2. Install requirements
3. Train model
4. Start backend
5. Open frontend

## Future Improvements
- Real-time smart meter integration
- Weather API integration
- User login system
- Monthly bill prediction
