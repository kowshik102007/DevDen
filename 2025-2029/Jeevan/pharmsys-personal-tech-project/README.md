# Pharmacy Management System 

## Overview
The Pharmacy Management System is a lightweight, web-based application designed to streamline daily pharmacy operations. It provides an intuitive interface to effectively manage medicine inventory, track stock levels, and generate insightful analytics for the business.

## Features 
- **Inventory Management:** Easily add, view, update, and remove medicines from the system.
- **Stock Tracking:** Monitor available quantities, expiration dates, and pricing details.
- **Analytics & Reporting:** Generate sales reports, calculate revenue, and analyze stock data.
- **User-Friendly Interface:** Clean, interactive dashboards built with modern web standards.
