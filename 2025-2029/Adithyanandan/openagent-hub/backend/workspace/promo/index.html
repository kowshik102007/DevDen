Self‑hosted AI Operating System

Unify multiple LLM providers, agents, tools, memory and RAG in a single Docker‑first platform.

★ Star on GitHub

Key Features

Multi‑Provider Support

Connect Groq, OpenRouter, Ollama, Google AI Studio, and any OpenAI‑compatible endpoint. Automatic routing, health checks and failover.

Agent Framework

Give a goal, watch the ReAct loop, tool calls and live step‑by‑step timeline. Supports multi‑agent parallel execution.

MCP Integration

Plug‑in stdio MCP servers to expose filesystem, database, browser, GitHub, Slack and more as first‑class tools.

Persistent Memory

User, project and conversation memory that agents can read/write across runs.

Unified Model Catalog

All models normalized with capability metadata (context, vision, reasoning, speed).

Docker‑First Deployment

One‑command `docker compose up -d --build` gets you a full stack (PostgreSQL, FastAPI, React) on any host.

Get Started in 2 Minutes

git clone https://github.com/AdithyanandanArun/openagent-hub.git
cd openagent-hub
docker compose up -d --build

Open http://localhost:3000 in your browser, register, and add your first provider.
