# Bellman Cartoon

## Overview

Bellman Cartoon is an interactive educational tool designed to help students understand Reinforcement Learning concepts through Value Iteration in a Gridworld environment.

The project visualizes how state values are updated using the Bellman Equation and allows users to observe the learning process step-by-step.

---

## Features

- Interactive Gridworld Environment
- Step-by-Step Bellman Updates
- Value Iteration Visualization
- Policy Arrow Display
- Convergence Tracking
- Adaptive Quiz System
- Educational Reinforcement Learning Simulator

---

## Environment Details

- Grid Size: 4 × 4
- Goal State Reward: +10
- Trap State Reward: -5
- Step Penalty: -0.1
- Discount Factor (γ): 0.9

### Actions

- Up
- Down
- Left
- Right
